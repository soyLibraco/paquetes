# Consigna de ejercicio

## Paquete
- Crear un paquete llamado `cliente`

## Módulos del paquete
- Crear dos módulos para el paquete:
    - Busqueda
    - Datos
-**`datos`**: contiene el diccionario variable base_datos
-**`busqueda`**: contiene una función que recibe el nombre de un cliente, itera sobre la lista de clientes y si el parámetro existe en la lista, entonces devolver el nombre del cliente. De lo contrario devolver None.

## Script principal
Crear un script principal llamado **`app`**.
Debe contener importaciones correspondientes. Una función `main()` que pregunte al usuario el nombre de un cliente a buscar.
Si el cliente es encontrado, mostrar su nombre, de lo contrario mostrar "Cliente no encontrado".
